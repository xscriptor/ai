# DevX Skill

Technical UI documentation for the DevX visual system used in this repository.

This skill is intended for AI-assisted design and implementation work on the `resources/vscode` page and related components.

## What it covers

- design intent and visual language
- theme variable usage
- responsive behavior
- component boundaries
- recommended folder structure
- accessibility and motion rules
- implementation guidance for future UI work
