<h1 align="center">@xscriptor/skill-samurai</h1>

<p>Security platform skill for the Samurai system. Contains security architecture, backend patterns, component conventions, database schema, and design token guidelines.</p>

<h2>Installation</h2>

<pre><code># Install to OpenCode (~/.config/opencode/skills/samurai/)
npx @xscriptor/skill-samurai

# Install to Claude Code (~/.claude/skills/samurai/)
npx @xscriptor/skill-samurai --anthropic

# Preview what will be installed
npx @xscriptor/skill-samurai --dry-run</code></pre>

<h2>Usage</h2>

<p>Once installed, the skill is loaded automatically when working on Samurai security projects. Invoke it manually with <code>/samurai</code> in Claude Code, or use <code>@samurai</code> in OpenCode.</p>

<p>The skill covers:</p>
<ul>
  <li><strong>Architecture</strong> — zero-trust design, API gateway, mTLS, audit logging</li>
  <li><strong>Backend patterns</strong> — repository pattern, middleware pipeline, circuit breaker</li>
  <li><strong>Component patterns</strong> — atomic design, accessibility-first (WCAG 2.1 AA)</li>
  <li><strong>Database schema</strong> — UUID v7, soft deletes, row-level security, audit columns</li>
  <li><strong>Design tokens</strong> — high contrast, color-blind safe, security UX semantics</li>
  <li><strong>Export patterns</strong> — barrel exports, public API surface conventions</li>
</ul>
