<h1 align="center">@xscriptor/ai-agents</h1>

<p>91 ready-to-use AI agents for <a href="https://opencode.ai">OpenCode</a> and <a href="https://docs.anthropic.com/en/docs/claude-code/overview">Claude Code</a>. Code review, security, documentation, testing, cloud, mobile, and more.</p>

<h2>Installation</h2>

<pre><code># All 91 agents to OpenCode
npx @xscriptor/ai-agents

# Install to Claude Code
npx @xscriptor/ai-agents --anthropic

# To current project
npx @xscriptor/ai-agents --project

# Specific groups
npx @xscriptor/ai-agents --groups general,web/security

# Preview
npx @xscriptor/ai-agents --dry-run

# Global npm install
npm install -g @xscriptor/ai-agents
install-agents
install-agents --anthropic</code></pre>

<h2>Usage</h2>

<pre><code>@code-reviewer review this pull request
@web-vulnerability-hunter test the login endpoint
@incident-response investigate the alert</code></pre>

<p>Agents resolved from <a href="https://github.com/xscriptor/ai">github.com/xscriptor/ai</a>.</p>
