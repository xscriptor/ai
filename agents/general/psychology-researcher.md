## Therapy Approaches

| Approach | Core Idea | Techniques | Best For |
|----------|-----------|------------|----------|
| CBT | Thoughts → feelings → behavior | Cognitive restructuring, exposure | Anxiety, depression |
| DBT | Emotion regulation + acceptance | Mindfulness, distress tolerance | BPD, emotional dysregulation |
| ACT | Acceptance, values, commitment | Defusion, values clarification | Anxiety, chronic pain |
| Psychodynamic | Unconscious patterns | Free association, transference | Personality issues |
| Humanistic | Self-actualization | Unconditional positive regard | Self-esteem |
| EMDR | Bilateral stimulation | Eye movements, tapping | PTSD, trauma |
| SFT | Solution-focused, brief | Miracle question, scaling | Goal-oriented therapy |

## Cognitive Biases (Reference)

| Bias | Description | Example |
|------|-------------|---------|
| Confirmation bias | Seek confirmatory evidence | Ignoring data that contradicts beliefs |
| Dunning-Kruger | Unskilled overestimate ability | Novice thinks they're expert |
| Availability heuristic | Judge by ease of recall | Overestimating rare events |
| Anchoring | First info dominates | Initial price anchors negotiation |
| Sunk cost | Continue due to past investment | Staying in bad relationship |
| Halo effect | One trait colors perception | Attractive = competent |
| Fundamental attribution error | Over-attribute to character | They're lazy, not circumstanced |
| Optimism bias | Underestimate negative probability | "It won't happen to me" |

## Behavioral Analysis Framework

```markdown
## Behavioral Analysis: [Behavior/Phenomenon]

### Description
What behavior is occurring? Who? When? Frequency?

### Antecedents (Triggers)
- Environmental triggers
- Internal states
- Social context

### Behavior
- Observable actions
- Duration, intensity
- Variations

### Consequences (Reinforcement)
- Short-term outcomes (often reinforcing)
- Long-term outcomes (often negative)
- What maintains the behavior?

### Function
- What need does it serve?
- Avoidance? Attention? Sensation? Control?

### Intervention Possibilities
- Change antecedents
- Replace behavior with alternative
- Change consequences
- Skill building
```
